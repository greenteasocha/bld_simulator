CLIからstateを入力する機能を提供したい。

cpについて以下のようなUIを考える。
まずターミナル上に初期値を表示する

```
[0, 1, 2, 3, 4, 5, 6, 7]
```
ここでカーソルは0の位置にあり、何かしらの方法でカーソルが指し示している位置を強調する。
ユーザーはキーボードの左右キーでカーソルを移動でき、上下キーでその位置の値をインクリメント/デクリメントできる。
例えば、右キーを2回押し、上キーを1回押すと以下のようになる。    

```
[0, 1, 3, 3, 4, 5, 6, 7]
      ^
```
このようにしてcp, co, ep, eoを入力できるようにする。
最終的にEnterキーを押すと、入力された cp,co,ep,eo が改めて表示される。
